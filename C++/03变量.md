### 打印

- C

```c
//
printf("你好\n");

printf("工资：%f  年假： %d", 33333.33,12); 
// 类似模板字符串
//%f 浮点型， %d整数, %s字符串
```

- c++

```c++
#include <iostream>  //头文件

int main(void){
    //std 命名空间
    //:: cout 命名空间内的cout对象
	std::cout << "1.网站404攻击" << std::endl;
}
```
